# Cloud Security Learning Tracker

A personal 6-month learning roadmap tracker for aspiring cloud security engineers. Track your daily progress through structured learning modules, manage study habits, and build your portfolio.

## 🎯 Features

- **Roadmap Tracker** - 180-day learning path broken down into months, weeks, and days
- **Dashboard** - Overview of progress, study streak, and daily goals
- **Daily Study Log** - Log your study sessions and track time commitment
- **Skill Tracker** - Auto-calculated skills based on completed roadmap items
- **Project Tracker** - Keep track of lab builds and portfolio projects
- **Notes Page** - Organize learning notes by category
- **Offline Support** - All data saved in localStorage, works completely offline

## 💾 Data Storage

All progress is stored in browser's localStorage:
- Roadmap progress
- Study logs
- Completed projects
- Learning notes
- Skill levels (auto-calculated)
